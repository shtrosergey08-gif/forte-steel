FORTE STEEL — Что снять на заводе завтра (шот-лист + сценарии)

Снимай вертикально (9:16), телефон горизонтально НЕ держи. Каждый клип 3–6 сек, снимай с запасом. Свет — у ворот/окон, не в тёмном углу. Цель: контент для Reels, сайта, Avito и «доверие = живой завод».

📸 ОБЯЗАТЕЛЬНЫЙ ШОТ-ЛИСТ (галочки)

Производство (для блока «Завод полного цикла» + доверие): - [ ] Цех общим планом — широкий проход, станки в ряд (медленная проводка телефоном) - [ ] Искры сварки крупным планом (замедление потом → эффектно) - [ ] Рука сварщика + шов вблизи - [ ] Станок плазменной/лазерной резки в работе (если есть) - [ ] Гибочный станок гнёт профиль - [ ] Камера порошковой покраски — деталь заезжает/окрашена - [ ] Готовая окрашенная графитовая деталь крупно (фактура RAL 7024) - [ ] Склад металла — ряды профтрубы (масштаб = серьёзность) - [ ] Рабочие в процессе (живость, каски/спецовка) - [ ] Логотип/вывеска Промэнерго, проходная

Детали для сайта/Avito (предметка): - [ ] Узел соединения каркаса крупно (качество сварки) - [ ] Сравнение: окрашенная vs неокрашенная деталь - [ ] RAL-веер в руке на фоне цеха - [ ] Образец кровли (профлист/алюмокомпозит) крупно

Для «движущегося» контента: - [ ] Таймлапс любого процесса (поставь телефон, снимай 1–2 мин → ускоришь) - [ ] Проезд камеры вдоль готовой конструкции

🎬 ГОТОВЫЕ СЦЕНАРИИ REELS (снять кадры → смонтировать потом)

Reel 1 — «Почему наш навес не сложится зимой» (доверие)

  • Хук (0–2 сек): текст на экране «Дешёвый навес зимой складывается. Вот почему наш — нет» + кадр толстого профиля/шва.
  • Тело: сварка → станок → камера покраски → готовая деталь.
  • Финал: текст «Считаем каркас под снег вашего региона по СП 20.13330. Завод полного цикла с 2012» + лого FORTE STEEL.
  • Подпись: Навес — это не крыша на палках. Это инженерная конструкция, которая держит тонны снега. Мы считаем её под ваш регион. #навес #навесpodключ #строительство

Reel 2 — «Завод, а не гаражная бригада» (отстройка)

  • Хук: «Вы платите бригаде — а навес варят в гараже. Мы делаем по-другому»
  • Тело: широкий план цеха → станки → склад металла → рабочие.
  • Финал: «Собственный завод. Свой металл, своя покраска, своя гарантия 5 лет».
  • Подпись: Разница между «купить навес» и «заказать у завода» — в том, кто отвечает за результат. #производство #металлоконструкции

Reel 3 — «От металла до графитового навеса» (процесс/эстетика)

  • Хук: ржавый профиль крупно → «Из этого…»
  • Тело: резка → сварка → покраска → «…получается это» → окрашенная графитовая деталь.
  • Финал: рендер готового навеса с сайта + лого.
  • Подпись: Путь одного навеса. Графит RAL 7024, заводская порошковая покраска. #дизайн #экстерьер

Reel 4 — «RAL 7024» (премиум-эстетика, короткий)

  • Только крупные планы: веер RAL → окрашенная деталь → фактура → лого. Музыка спокойная, премиум.
